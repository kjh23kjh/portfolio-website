# Simple Portfolio Website

This is a simple responsive portfolio website made with HTML, CSS, and basic JavaScript.

## Pages

- `index.html`: Self-introduction page
- `activities.html`: Activity list page

## How to Use

1. Open `index.html` in your browser.
2. Edit `Your Name`, `your.email@example.com`, and activity text with your own information.
3. Keep all files in the same folder so the CSS and JavaScript load correctly.

## Files

```text
portfolio-website/
  index.html
  activities.html
  style.css
  script.js
  README.md
```
