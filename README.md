# Simple Portfolio Website

This is a simple responsive portfolio website made with HTML, CSS, and basic JavaScript.

## Pages

- `index.html`: Self-introduction page
- `activities.html`: Activity list page
- AI survey service form powered by a Make.com webhook

## How to Use

1. Open `index.html` in your browser.
2. Edit `Your Name`, `your.email@example.com`, and activity text with your own information.
3. In `script.js`, replace `https://hook.us1.make.com/your_unique_webhook_id` with your Make.com Custom Webhook URL.
4. Keep all files in the same folder so the CSS and JavaScript load correctly.
